# Metaheuristic Optimized K-Medoids Clustering

## Description

This Shiny application implements K-Medoids clustering optimized using:

- PAM
- Genetic Algorithm (GA)
- Particle Swarm Optimization (PSO)
- Cuckoo Search Algorithm (CSA)
- Grey Wolf Optimizer (GWO)

## Features

- CSV Upload
- Data Preprocessing
- Mahalanobis Outlier Detection
- K-Medoids Clustering
- Silhouette Analysis
- Validation Metrics
- Visualization
- Download Results
